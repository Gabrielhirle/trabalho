introduçâo do HTML

O html e o linguagem padrâo utilizadas para criar paginas da web permitem que os desenvolvedores estruturem no conteudo de uma pagina como textos ,imagens, links e outros elementos usando uma serie de tags que definem a funçâo de cada elemento na pagina

O que sâo tags

As tags sâo elementos fundamentais que sâo usados para marcar estruturas o conteudo de uma pagina da web. As tags sâo representadas por pares de colchetes angulares("<>" e () e sâo usadas para indicar ao navegador da web como o conteudo deve ser exibido

exemplo de tags de lista

  • lista ordenada ( < ol> )
  • lista desordenada ( < ul> )
  • lista de definiçâo (< dl> ) (< dt> ) (< dd> )

outras tags

  • tags de link (< a href="">)
  • tags de imagens (< img src="" alt="">)
  • tags header ( < header> )
  • tags div ( < div> )
  • tags section ( < section> )
  • tads main ( < main> )
  • tags aside ( < aside> )
  • tags nav ( < nav> )
  • atributo id ( < p id=""> )
  • atributo class ( < p class=""> )

introduçâo CSS

O css e uma linguagem de estilo usada para controlar a aprensantaçâ0 e o layout de elementos html em uma pagina da web. Com o css voce pode definir a aparencia de elementos, como cores, fontes tamanhos, margens,espaçamentos e muito mais

Alguns atributos que são usadas no css

  • atributo id ( < p id=""> )
  • atributo class ( < p class=""> )

o atributo id e usado para dar um identificaçâo unica a um elemento html e so um elemento dev e conter esse (id)

O atributo class e usado para aplicar estilos comuns a um ou mais elementos html voce pode reutilizar a mesma classe em varios elementos da pagina

seleção de cores

As cores podem ser colocados tanto no fundo da pagina cor de fonte, cor da borda

E essas cores podem estar disponiveis em em rgb, hexadecimal e rgba

exemplo de como estão disponiveis as cores

cores em rgb
  • preto (rgb (0, 0, 0))
  • branco (rgb (255, 255, 255))
cores em hexadecimal
  • preto (#000000)
  • branco (#ffffff)
cores/ transparencia em rgba
  • transparente (255, 99, 71, 0)
  • transparencia (255, 99, 71, 1)
